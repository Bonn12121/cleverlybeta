Welcome to Cleverly

You need a free OpenRouter API key to chat. Takes 1 minute — no credit card needed.

  1. Go to openrouter.ai and click Sign Up (it's free)
  2. After signing in, click your avatar → API Keys
  3. Click Create Key, give it any name, copy it
  4. Paste your key below and hit Start Chatting

🔒 Your key is held in memory only — never saved to disk, never sent anywhere except OpenRouter's servers.
